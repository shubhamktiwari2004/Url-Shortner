# SnapURL

A secure and scalable URL shortening service built with **Java Spring Boot** that transforms long URLs into short, shareable links. SnapURL provides fast URL redirection, click tracking, and user-specific URL management using **JWT Authentication**.

---

## Features

- Shorten long URLs into unique short links
- Fast URL redirection
- JWT-based authentication and authorization
- User-specific URL management
- Click count tracking
- RESTful API architecture
- MySQL database integration
- Layered architecture following Spring Boot best practices

---

## Tech Stack

- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)
- Spring Data JPA (Hibernate)
- MySQL
- Maven
